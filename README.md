# ইসলামিক কোটেশন জেনারেটর 🌙

বাংলা ভাষায় ইসলামিক উক্তি বা কোটেশন প্রদর্শনের জন্য একটি ইন্টার‍্যাক্টিভ ওয়েব অ্যাপ।  
এটি র‍্যান্ডমভাবে কুরআন, হাদিস এবং ইসলামিক মনীষীদের অনুপ্রেরণামূলক বাণী দেখায়।  
প্রকল্পটি সম্পূর্ণরূপে HTML, Tailwind CSS এবং JavaScript দিয়ে তৈরি।

---

## ফিচারসমূহ

- 🎯 র‍্যান্ডম ইসলামিক কোটেশন (বাংলা ভাষায়)
- 📝 লেখকের নাম সহ কোট প্রদর্শন
- 🎨 Tailwind CSS দিয়ে আধুনিক ও মোবাইল-ফ্রেন্ডলি ডিজাইন
- 😊 ইন্টার‍্যাক্টিভ ইমোজি রিএকশন
- 🌙 গুগল ফন্ট ইন্টিগ্রেশন (Hind Siliguri, Baloo Da 2)
- 🕋 ইসলামিক ভাবমূর্তি ও অনুপ্রেরণা

## কিভাবে রান করবেন

1. **রিপোজিটরি ক্লোন করুন:**
   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```

2. **প্রজেক্ট ফোল্ডারে যান:**
   ```bash
   cd your-repo
   ```

3. **index.html ফাইল ওপেন করুন যেকোনো ব্রাউজারে।**
